# 누적 합

- 배열의 일부 구간에 대한 합을 빠르게 구할 수 있게 해주는 스킬

- N 개의 원소로 이루어진 배열에서 반복문으로 하면 **O(N)** 이 걸리지만 부분합을 이용하면 **O(1)** 이 걸린다.

#### 1차원 에서는

초기화 `S[i]=S[i-1]+A[i]`

찾을 때 `[i,j] => S[j]-S[i-1]`

#### 2차원 에서는

초기화 : `S[i][j]=arr[i][j]+S[i-1][j]+S[i][j-1]-S[i-1][j-1]`

찾기 : `[(x1,y1),(x2,y2)] => S[x2][y2]-S[x1-1][y2]-S[x2][y1-1]+S[x1-1][y1-1]`
